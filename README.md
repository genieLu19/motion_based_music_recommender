## 🎧Motion-Based Music Recommender
**Интеллектуальная система подбора музыкальных треков по ритму движения пользователя.**

Проект анализирует данные движения пользователя (акселерометр) и подбирает музыкальные композиции, темп которых соответствует активности человека. Система использует методы обработки сигналов и нейросетевую модель для оценки соответствия музыкальных треков ритму движения.
Проект разработан в рамках выпускной квалификационной работы.

## 📌Основная идея
Система выполняет следующие шаги:
1. Анализирует данные движения пользователя.
2. Вычисляет темп активности (BPM).
3. Анализирует характеристики музыкальных треков.
4. Оценивает соответствие треков ритму движения.
5. Формирует список рекомендованных треков.

## ⚙️Архитектура системы
▪️ Система состоит из нескольких взаимосвязанных модулей.

▪️ Сначала данные движения пользователя загружаются из датасета MotionSense. Эти данные представляют собой временные ряды акселерометра и используются для анализа активности пользователя.

▪️ Затем модуль обработки сигналов выполняет фильтрацию данных и вычисляет темп движения пользователя (BPM).

▪️ Параллельно модуль обработки музыкальных данных загружает базу треков и извлекает их характеристики, такие как BPM, энергичность и танцевальность.

▪️ После этого нейросетевая модель MatchNet анализирует параметры движения пользователя и характеристики музыкальных треков и вычисляет степень соответствия каждого трека текущему ритму активности.

▪️ На основе полученных оценок модуль рекомендаций ранжирует треки и формирует список наиболее подходящих музыкальных композиций.

▪️ Результаты работы системы отображаются в веб-интерфейсе, реализованном с помощью Streamlit, где пользователь может выбрать параметры и получить список рекомендованных треков.

## 🖥️ Интерфейс пользователя
Взаимодействие пользователя с системой реализовано через веб-интерфейс, созданный с использованием библиотеки Streamlit.

Через интерфейс пользователь может:

▪️ выбрать тип активности (например, ходьба или бег);

▪️ указать музыкальный жанр;

▪️ задать предпочтительных или исключённых исполнителей;

▪️ выбрать стиль музыки (более энергичная или более спокойная);

▪️ указать количество треков для рекомендаций.

После выбора параметров пользователь запускает анализ.  
Система вычисляет BPM движения, анализирует музыкальную базу и выводит список треков, наиболее подходящих текущему ритму активности.
Результаты отображаются в виде таблицы с основными характеристиками треков.

## ✅Запуск проекта
**1. Клонировать репозиторий:**

git clone https://github.com/USERNAME/motion_based_music_recommender.git
cd motion_based_music_recommender

**2. Установить зависимости:**

pip install -r requirements.txt

**3. Запустить приложение:**

streamlit run full_system.py
После запуска откроется веб-интерфейс системы.

## 📈 Результаты работы модели
Для оценки качества работы нейросетевой модели MatchNet были использованы стандартные метрики регрессии.
Модель оценивает степень соответствия музыкального трека ритму движения пользователя и возвращает значение в диапазоне от 0 до 1, где более высокие значения означают лучшее соответствие.
В ходе тестирования были рассчитаны следующие метрики:

▪️ **MSE (Mean Squared Error)** — среднеквадратичная ошибка предсказаний;

▪️ **MAE (Mean Absolute Error)** — средняя абсолютная ошибка;

▪️ **R² (коэффициент детерминации)** — показывает, какую долю вариации данных объясняет модель;

▪️ **коэффициент корреляции Пирсона** — отражает степень связи между предсказанными и истинными значениями.

Полученные результаты показывают, что модель способна выявлять зависимость между ритмом движения пользователя и характеристиками музыкальных треков, что позволяет формировать корректные рекомендации.
